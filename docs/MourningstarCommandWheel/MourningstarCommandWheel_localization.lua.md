# MourningstarCommandWheel_localization.lua - Документация

## Описание файла

Файл локализации мода, содержащий все текстовые строки для разных языков.

## Поддерживаемые языки

- `en` - английский
- `ru` - русский

## Локализованные строки

### `mod_name`
Название мода.

**Значения:**
- `en`: "Mourningstar Command Wheel"
- `ru`: "Колесо команд Морнингстар"

### `mod_description`
Описание мода.

**Значения:**
- `en`: "A convenient radial menu system for quick access to all hub menus and features."
- `ru`: "Удобная система радиального меню для быстрого доступа ко всем меню и функциям хаба."

### `open_command_wheel_key`
Название настройки для клавиши открытия колеса.

**Значения:**
- `en`: "Open Command Wheel"
- `ru`: "Открыть колесо команд"

### `open_command_wheel_key_description`
Описание настройки для клавиши открытия колеса.

**Значения:**
- `en`: "Key to open the command wheel."
- `ru`: "Клавиша для открытия колеса команд."

## Использование

Локализация используется через `mod:localize(key)`, где `key` - это ключ локализации (например, `"mod_name"`).

## Добавление новых языков

Для добавления нового языка необходимо:
1. Добавить поле с кодом языка (например, `de` для немецкого)
2. Добавить переводы для всех ключей локализации

## Примечания

Мод также использует игровую локализацию для названий кнопок (ключи начинающиеся с `"loc_"`), которые обрабатываются через функцию `Localize()` игры.
