# FriendlyFireDamage

Мод для Warhammer 40,000: Darktide, который показывает уведомления при получении урона от сокомандников (friendly fire).

## Возможности

- Отслеживание урона от сокомандников (оружие, способности)
- Отслеживание урона от взрывов бочек (barrels)
- Уведомления при получении урона с указанием имени игрока и количества урона
- Отображение общего урона от каждого игрока в уведомлении (опционально)
- Настройка минимального порога урона для уведомлений

## Установка

1. Скопируйте папку `FriendlyFireDamage` в директорию модов Darktide
2. Запустите игру
3. Мод будет автоматически активирован

## Настройки

- **Minimum damage threshold** - Минимальный порог урона для показа уведомления (0-100). Уведомления будут показываться только если урон превышает это значение.

- **Show total damage in notification** - Показывать общий урон от игрока в уведомлении (в скобках)

## Отслеживаемые источники урона

Мод отслеживает урон от следующих источников:

1. **Прямой урон от оружия** - выстрелы, удары оружием
2. **Урон от способностей** - взрывы, эффекты способностей
3. **Урон от взрывов бочек** - когда сокомандник взрывает бочку, которая наносит урон вам

## Примечания

- Статистика сбрасывается при начале новой миссии
- Урон отслеживается только для живых игроков
- Мод работает только в игровых сессиях (не в хабе)
