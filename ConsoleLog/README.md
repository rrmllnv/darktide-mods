# ConsoleLog

Мод для отображения логов других модов на экране игры.

## Описание

ConsoleLog позволяет другим модам выводить логи прямо на экран игры, вместо использования консоли. Логи отображаются в виде текстовой панели на половине экрана.

## Использование в других модах

```lua
local console_log_mod = get_mod("ConsoleLog")

-- Добавить лог
console_log_mod:add_log("MyMod", "Текст лога")

-- Добавить лог с цветом
console_log_mod:add_log("MyMod", "Текст лога", {255, 255, 0, 0}) -- красный

-- Очистить все логи
console_log_mod:clear_logs()

-- Включить/выключить отображение
console_log_mod:set_enabled(true)
```

## Настройки

- **Включен** - включить/выключить отображение логов
- **Максимум строк** - максимальное количество строк логов (5-50)
- **Размер шрифта** - размер шрифта (10-24)
- **Позиция** - позиция панели на экране (верхний левый, верхний правый, нижний левый, нижний правый)
- **Прозрачность фона** - прозрачность фона панели (0-255)
