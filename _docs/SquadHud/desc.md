[b][size=4]What it is and why use it[/size][/b]

Squad HUD replaces the vanilla squad panels with four compact panels for the whole strike team. Each panel can show class/status icon, player name, total character level, teammate distance or status text, combat ability state, inventory icons, grenade/blitz and ammo values, expedition salvage, toughness, health, wounds, and active health/toughness numbers.

It is for players who want the squad state to be easier to read at a glance: who is alive, down, dead, trapped, carrying an objective item, in coherency, low on health, or missing resources.


[b][size=3]How to use[/size][/b]
[list=1]
[*]Install the mod (Darktide Mod Framework / your mod manager, following the game's usual mod setup).[/*]
[*]Enable the mod and open its options in the mod menu.[/*]
[*]Adjust position, opacity, and scale in the Display group.[/*]
[*]Use Visible panels to show the full squad, only yourself, or only teammates (teammates-only mode keeps the same visual anchor by packing teammate panels into the lower slots).[/*]
[*]Use Local player order to place your own panel at the top or bottom when your panel is visible.[/*]
[*]Use the Expanded View group to pick Compact vs Expanded default layout, bind Show expanded view, choose Toggle or Hold for the key, and optionally show the key hint.[/*]
[*]Use the Squad HUD group to turn individual HUD parts on or off.[/*]
[*]Use the Vanilla HUD group if you want to hide the original player panels and avoid duplicate UI.[/*]
[*]Changes are stored in DMF's saved settings; use Reset settings to restore the mod's built-in defaults.[/*]
[/list]
[b]Benefits[/b]
[list]
[*]One compact four-slot squad panel, with optional top or bottom placement for the local-player panel.[/*]
[*]Teammate panels include class/status icon, name, distance/status text, health, toughness, and wounds.[/*]
[*]Long names can animate inside the name area instead of overflowing.[/*]
[*]Optional player-colored names, with inactive gray styling for bots, dead players, and rescued/hogtied players.[/*]
[*]Optional total level next to player names, calculated from character progression data exposed by the game backend.[/*]
[*]Inventory icons show grenade/blitz, ammo state, medical crate, ammo crate, stimms, and expedition salvage when available.[/*]
[*]Grenade/blitz and ammo values can be hidden, always shown, or shown for 5 seconds after they change.[/*]
[*]Ammo value can be displayed as a percentage, as ammo count only, or as current / max ammo (default).[/*]
[*]Stimm icons are recolored by syringe type using the mod's built-in colors.[/*]
[*]Health/toughness values use an active-bar display with priority for health changes and a timed return to health.[/*]
[*]Overshield uses a golden toughness color and a short peel-off effect when the overshield ends.[/*]
[*]Toughness damage adds a compact pulse around the panel, with a stronger armor-break pulse when toughness reaches zero.[/*]
[*]Expanded View key shows account names, platform icons, and extra squad HUD details; optional hint text under the last panel in missions and the Psykhanium.[/*]
[*]Optional hiding of vanilla local-player and teammate panels.[/*]
[/list]

[b][size=4]Coherency[/size][/b]

Squad HUD shows coherency information inside the teammate panel. For every teammate, the mod checks whether that player is in coherency with the local player and can show the distance to that teammate directly in the name block.

[list]
[*]Distance is shown as meters to the teammate and is capped to a compact value so it fits inside the panel.[/*]
[*]The distance text is hidden when a higher-priority player status is shown, such as downed, dead, netted, pounced, grabbed, or carrying a rescue status.[/*]
[*]Coherency color: green border means the teammate is in coherency with you.[/*]
[*]Coherency color: red border means the teammate is outside coherency.[/*]
[*]If Meters to teammate is turned off, the nickname takes the freed space and long-name marquee animation recalculates for the wider name area.[/*]
[/list]

[b][size=4]Settings by group[/size][/b]

[b][size=3]Display[/size][/b]
[list]
[*]Enable Squad HUD - on/off. Default on. Shows or hides the custom four-slot squad panel.[/*]
[*]Visible panels - dropdown. Default Full squad. Controls which Squad HUD panels are shown: Full squad, Only me, or Only teammates. In teammates-only mode, teammate panels are packed into the lower slots so the layout keeps the same anchor.[/*]
[*]Local player order - dropdown. Default Bottom. Controls whether your own panel is placed at the top or bottom when your panel is visible.[/*]
[*]Position, opacity & scale
[list=1]
[*]Offset X - horizontal offset from the top-left screen area in logical pixels, range 0 to 1920. Default 50.[/*]
[*]Offset Y - vertical offset from the top-left screen area in logical pixels, range 0 to 1080. Default 780.[/*]
[*]Opacity - 0.1 to 1.0. Default 1.0.[/*]
[*]Scale - 0.5 to 2.0. Default 0.8. Scale multiplier for the whole Squad HUD.[/*]
[/list]
[/*]
[/list]
[b][size=3]Expanded View[/size][/b]
[list]
[*]Expanded squad view - dropdown. Default Expanded. Options: Compact, Expanded. Controls how much extra squad HUD information is visible by default (before using the key).[/*]
[*]Show expanded view - keybind. Default Left Ctrl. Shows account names, platform icons, and additional squad HUD details according to the selected key mode. When expanded view is active, distance text is hidden so names can use the full name area.[/*]
[*]Expanded view key mode - dropdown. Default Hold. Toggle switches expanded view on/off; Hold shows it only while the key is held.[/*]
[*]Show Expanded View key hint - on/off. Default on. Brief hint under the last Squad HUD panel with your Expanded View key (or a reminder if no key is bound). Only in missions and the Psykhanium, not the Mourningstar hub.[/*]
[/list]
[b][size=3]Squad HUD[/size][/b]
[list]
[*]Show class icon - on/off. Default on. Shows the player class icon, or a status icon when the player is in a special state.[/*]
[*]Show ability icon - on/off. Default on. Shows the teammate combat ability state.[/*]
[*]Show player total level - on/off. Default on. Shows the player's total level next to the nickname. The total level is calculated from character progression data exposed by the game backend.[/*]
[*]Use player colors for names - on/off. Default on. Uses each player's assigned player color for normal names while keeping warning/status colors for temporary status text.[/*]
[*]Coherency
[list=1]
[*]Show meters to teammate - on/off. Default on. Shows distance in the name block. When off, the name area becomes wider and the name marquee recalculates for the larger area.[/*]
[/list]

[*]Items

[/list]
[*]Show ammo icon - on/off. Default on.[/*]
[list=1]
[*]When to show ammo value - dropdown. Default When changed. Options: Never, Always, When changed.[/*]

[/list]
[*]Show stimm icon - on/off. Default on.[/*]
[*]Show med crate icon - on/off. Default on.[/*]
[*]Show ammo crate icon - on/off. Default on.[/*]
[/list]
