[b][size=4]What it is and why use it[/size][/b]

Division HUD adds a separate combat interface in the style of [i]The Division[/i]: health, resources, and loadout are shown in one block. It’s for players who want a different look and a clearer, more unified picture in combat—without having to rebuild the entire vanilla HUD by hand.


[b][size=3]How to use[/size][/b]
[list=1]
[*]Install the mod (Darktide Mod Framework / your mod manager, following the game’s usual mod setup).[/*]
[*]Enable the mod and open its options in the mod menu.[/*]
[*]Adjust position and opacity as you like; turn off parts of the mod HUD if you want, and use the vanilla-hide toggles if you don’t want duplicate UI.[/*]
[*]Changes are stored in DMF’s saved settings; use Reset settings to restore the mod’s built-in defaults.[/*]
[/list]
[b]Benefits[/b]
[list]
[*]One readable combat status block in the chosen style.[/*]
[*]Flexible placement relative to screen center and adjustable opacity.[/*]
[*]You can disable individual bars/timer on the mod HUD.[/*]
[*]Optional hiding of selected vanilla HUD pieces (vanilla stamina and dodge are on by default so they stay hidden and don’t overlap the mod bars).[/*]
[*]Optional “dynamic” HUD shift tied to camera movement, with tunable strength and return speed.[/*]
[/list]

[b][size=3]Display[/size][/b]
[list]
[*]Show Division HUD — on/off. Default on. Hides or shows the entire Division HUD overlay.[/*]
[*]Toggle HUD visibility — keybind. Same as the Show Division HUD checkbox.[/*]
[*]Auto visibility switch
[list=1]
[*]First person — on/off. Default on. Show Division HUD in first person.[/*]
[*]Third person — on/off. Default on. Show Division HUD in third person.[/*]
[*]Device in hand — on/off. Default off. Off hides Division HUD while holding a device; on keeps it visible.[/*]
[/list]
[/*]
[*]Position, opacity & scale
[list=1]
[*]Offset X — horizontal offset from screen center in logical pixels (1920×1080 reference), range -960 to 960; negative = left, positive = right. Default 400.[/*]
[*]Offset Y — vertical offset from screen center in logical pixels (1920×1080 reference), range -540 to 540; negative = up, positive = down. Default 200.[/*]
[*]Opacity — 0.1 to 1.0. Default 1.0.[/*]
[*]Scale — 0.5 to 2.0. Default 0.8. Scale multiplier for the whole Division HUD.[/*]
[/list]

[b][size=3]Combat HUD (mod overlay)[/size][/b]
[list]
[*]Stamina and dodge bar — on/off. Default on.[/*]
[*]Toughness bar — on/off. Default on.[/*]
[*]Health bar — on/off. Default on.[/*]
[*]Ability bar / timer — on/off. Default on.[/*]
[*]Ability icon — on/off. Default on.[/*]
[*]Color pocket stimm icon — on/off. Default on. When enabled, the pocket stimm icon in the Division HUD strip is tinted by syringe type (speed, power, corruption cleanse, ability boost, broker syringe, etc.) using built-in colors, or RecolorStimms colors if that integration is on. When off, the icon uses the default white tint.[/*]
[*]Ammo text color by remaining fraction — on/off. Default on. On: the large clip and reserve numbers change color based on how much ammo you still have. When you have no ammo, those numbers keep the normal HUD text look but are slightly faded so an empty weapon is easy to spot. Off: the numbers always use the standard main color; when empty they are still slightly faded for clarity.[/*]
[*]Grenade slot color by charges — on/off. Default on. On: the grenade icon and charge counter change color by how many charges are left. With no charges left, the icon and counter are slightly faded in a neutral way. Off: the slot uses the usual white icon and default counter styling.[/*]
[*]Current weapon icon colors — on/off. Default on. On: tints the wielded-weapon strip icon for weapons included in this mod’s internal list by inactive, active, and optional cooldown states. Other weapons stay full white. Off: default strip icon styling.[/*]
[*]Buff rows — on/off. Default on. Shows active player buffs in rows below the stamina/dodge bars, up to 15 buffs total, 5 per row.[/*]
[*]Weapon & ammo slots background — dropdown. Default Weapon fill only. Options: Weapon fill only, Weapon fill + frame, Terminal gradient green, Terminal gradient black.[/*]
[*]Detected items area background — dropdown. Default Terminal gradient green. Options: Weapon fill + frame, Terminal gradient green, Terminal gradient black.[/*]
[/list]

[*]Max offset — 8–220. Default 100.[/*]
[*]Freeze when aiming — on/off. Default on. Freezes Dynamic HUD movement while aiming down sights; works with mouse and controllers.[/*]

[b][size=3]Alerts[/size][/b]
[list]
[*]Enable approach alerts — on/off. Default on. On: the mod may show alert rows on the Division HUD when a boss or specialist you allow in the lists below is detected. Off: those approach alerts do not appear.[/*]
[*]Maximum simultaneous alerts — 1–5. Default 2. This is how many alert rows can show at the same time. If more alerts happen while the screen is full, extras wait briefly and appear as a slot frees up.[/*]
[*]Alert duration (seconds) — 1–60. Default 6. Each alert row stays on screen about this many seconds, then disappears, or sooner if many alerts push it out of the visible stack.[/*]
[*]Alert display time bar — on/off. Default on. On: each row has a thin bar that shrinks to show how much of that row’s time is left. Off: no bar on the row.[/*]
[*]Bosses
[list=1]

[b][size=3]Integrations[/size][/b]
[list]
[*]Custom HUD — on/off. Default off. When enabled and Custom HUD has a saved layout for this HUD, Division HUD applies that position every frame and adds Dynamic HUD motion on top.[/*]
[*]StimmCountdown — on/off. Default on. When enabled and the StimmCountdown mod is loaded and enabled, the pocket stimm slot in the Division HUD strip shows that mod’s countdown (active effect / cooldown) using its own display options; when off, the slot only shows the usual 01/00 counter.[/*]
[*]RecolorStimms — on/off. Default off. When enabled and the RecolorStimms mod is loaded and enabled, pocket stimm icon colors are taken from that mod; when off or if that mod isn’t available, Division HUD falls back to its built-in syringe colors (when “Color pocket stimm icon” is on).[/*]
