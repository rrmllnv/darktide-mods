## Рекомендации

### Топ-5 вариантов:

1. **TacticalCommandWheel** - Прямолинейный, понятный, военный стиль
2. **VoxCommandWheel** - В тематике Warhammer 40k (Vox - система связи)
3. **SquadCommsWheel** - Краткий, военный, понятный
4. **BattleCryWheel** - Агрессивный, запоминающийся
5. **TacticalCommsWheel** - Тактический, военный

### Учитывая контекст:

- **Vox** - это система связи в Warhammer 40k, поэтому названия с "Vox" очень подходят
- **Tactical** - подчеркивает тактический аспект команд
- **Squad** - акцент на командной работе
- **Command** - прямо указывает на команды

### Финальная рекомендация:

**TacticalCommandWheel** или **VoxCommandWheel** - оба названия хорошо отражают суть мода и подходят для Warhammer 40,000: Darktide.
