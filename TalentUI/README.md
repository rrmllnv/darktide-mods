# TalentUI

Мод для Warhammer 40,000: Darktide, который добавляет отображение иконок способностей с таймерами кулдауна в team HUD.

## Функциональность

### Для тимейтов:
- **Иконка способности** - отображается слева от рамки здоровья каждого тимейта
- **Визуальное состояние** - иконка меняет цвет в зависимости от состояния:
  - Активна (зеленый)
  - На кулдауне (серый)
  - Без зарядов (красный)
- **Прогресс кулдауна** - анимированная заливка иконки показывает прогресс восстановления
- **Счетчик кулдауна** - текстовое отображение оставшегося времени или процента

### Для локального игрока:
- **Счетчик кулдауна** - текст поверх иконки способности в стандартном HUD
- **Два формата** - время в секундах или процент

## Настройки

- **Show Teammate Ability Icons** - показывать иконки способностей тимейтов
- **Show Teammate Ability Cooldown** - показывать текст кулдауна для тимейтов
- **Show Local Player Ability Cooldown** - показывать кулдаун для локального игрока
- **Cooldown Display Format** - формат отображения (вAW ремя/процент)
- **Ability Icon Size** - размер иконки способности (30-100)
- **Cooldown Text Font Size** - размер шрифта текста кулдауна (10-30)

## Техническая информация

Мод использует:
- `ability_extension` для получения данных о способностях
- `unit_data_extension` для чтения компонента `combat_ability`
- Систему цветов из `HudElementPlayerAbilitySettings`
- Hook'и для `HudElementTeamPlayerPanel` и `HudElementPlayerAbility`

## Совместимость

Мод совместим с:
- NumericUI
- Другими модами, изменяющими team HUD
