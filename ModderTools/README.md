# ModderTools

Инструменты для моддеров Darktide, позволяющие заменять реальные имена игроков на случайные для создания безопасных скриншотов.

## Особенности

- **Замена имен игроков** на случайные в миссиях
- **Детерминированная генерация** - каждый игрок всегда получает одно и то же случайное имя
- **Поддержка нескольких UI элементов:**
  - Панель команды (team panel)
  - Плавающие неймплейты (nameplates)
- **Автоматическая очистка кэша** при выходе из миссии
- **Безопасность** - не влияет на реальные данные игроков

## Установка

1. Скопируйте папку `ModderTools` в директорию модов игры
2. Перезапустите игру или перезагрузите моды

## Настройки

Мод имеет следующие настройки в меню модов:

- **Включить случайные имена** - основная настройка для активации замены имен
- **Замена имен в панели команды** - включить замену в панели тимейтов
- **Замена имен в неймплейтах** - включить замену в плавающих неймплейтах

## Как это работает

1. В миссиях реальные ники игроков заменяются на случайные из предустановленного списка
2. Каждому игроку по account_id генерируется уникальное случайное имя
3. Имена кэшируются для консистентности в течение сессии
4. При выходе из миссии кэш автоматически очищается

## Список случайных имен

Мод использует 100 уникальных имен, таких как:
- ShadowHunter, VoidWalker, StormBringer
- BloodReaper, IronFist, SoulKeeper
- DeathDealer, FlameBearer, IceGuardian
- И многие другие...

## Совместимость

Мод совместим с:
- TalentUI
- NumericUI
- Другими модами, модифицирующими UI игроков

## Безопасность

Мод не сохраняет никаких реальных данных игроков и не передает их куда-либо. Все изменения происходят только локально в интерфейсе игры.
