# HudElementTeamPlayerPanel - Виджеты

Виджеты в панели `HudElementTeamPlayerPanel` (из файла `hud_element_team_player_panel_definitions.lua`):

## Все виджеты в панели:

1. **`player_name`** — имя игрока (текст)
2. **`rich_presence`** — статус игрока (текст)
3. **`respawn_timer`** — таймер возрождения (текст)
4. **`throwable`** — иконка метательного оружия (текстура)
5. **`ammo_status`** — статус боеприпасов (текстура)
6. **`pocketable`** — карманный предмет (текстура)
7. **`pocketable_small`** — маленький карманный предмет (текстура)
8. **`voice_indicator`** — индикатор голоса (текстура)
9. **`coherency_indicator`** — индикатор когерентности (текстура) — используется как точка отсчета для способностей
10. **`panel_background`** — фон панели (текстура)
11. **`toughness`** — полоса стойкости (текстура)
12. **`toughness_ghost`** — призрачная полоса стойкости (текстура)
13. **`toughness_bar_background`** — фон полосы стойкости (текстура)
14. **`player_icon`** — иконка игрока/портрет (текстура)
15. **`status_icon`** — иконка статуса (текстура)

## Дополнительно:

- **`health_bar_segment_definition`** — определение для сегментов здоровья (создаются динамически, не виджет)
