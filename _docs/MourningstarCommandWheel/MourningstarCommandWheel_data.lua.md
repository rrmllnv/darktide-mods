# MourningstarCommandWheel_data.lua - Документация

## Описание файла

Файл данных мода, определяющий метаданные мода для системы настроек DMF (Darktide Mod Framework).

## Структура данных

Файл возвращает таблицу с следующими полями:

### `name`
Имя мода для отображения в настройках. Использует локализацию через `mod:localize("mod_name")`.

### `description`
Описание мода для отображения в настройках. Использует локализацию через `mod:localize("mod_description")`.

### `is_togglable`
Флаг, указывающий, можно ли включать/выключать мод. Установлен в `false` - мод всегда активен.

### `options`
Таблица с настройками мода.

#### `options.widgets`
Массив виджетов настроек.

##### Виджет `open_command_wheel_key`
Настройка для назначения клавиши открытия колеса команд.

**Поля:**
- `setting_id` - `"open_command_wheel_key"` - идентификатор настройки
- `type` - `"keybind"` - тип виджета (keybind)
- `default_value` - `{}` - значение по умолчанию (пустой массив = клавиша не назначена)
- `keybind_trigger` - `"held"` - триггер срабатывания (удержание клавиши)
- `keybind_type` - `"function_call"` - тип keybind (вызов функции)
- `function_name` - `"command_wheel_held"` - имя функции, вызываемой при удержании клавиши

## Использование

Этот файл загружается автоматически системой DMF при инициализации мода. Настройки отображаются в меню настроек модов игры.

## Связанные функции

- `mod:command_wheel_held()` - вызывается при удержании назначенной клавиши
- `mod:_is_command_wheel_key_pressed()` - проверяет состояние клавиши
- `mod.on_setting_changed()` - обрабатывает изменения настроек
