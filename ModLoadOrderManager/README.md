# Mod Load Order Manager

Программа для управления порядком загрузки модов Warhammer 40,000: Darktide.

## Описание

Приложение позволяет легко включать и выключать моды, редактируя файл `mod_load_order.txt`. Моды, начинающиеся с `--`, считаются отключенными, остальные - активными.

## Требования

- Python 3.7 или выше
- Tkinter (обычно входит в стандартную установку Python)

## Установка

1. Убедитесь, что Python установлен на вашем компьютере
2. Скачайте или склонируйте проект
3. Запустите `main.py`

## Использование

1. Запустите `main.py`
2. Укажите путь к файлу `mod_load_order.txt` (по умолчанию используется стандартный путь Steam)
3. Используйте чекбоксы для включения/выключения модов
4. Нажмите "Сохранить" для применения изменений

## Функции

- Просмотр всех модов из файла `mod_load_order.txt`
- Включение/выключение модов через чекбоксы
- Массовое включение/выключение всех модов
- Автоматическое сохранение заголовка файла
- Валидация и обработка ошибок

## Расположение файла

По умолчанию программа ищет файл по пути:
```
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\mods\mod_load_order.txt
```

Вы можете указать другой путь через кнопку "Обзор..." или вручную в поле пути.

## Запуск

### Вариант 1: Через bat-файлы (рекомендуется)

- **`run.bat`** - основной скрипт запуска с проверкой ошибок
- **`start.bat`** - простой вариант запуска
- **`LAUNCH.bat`** - запуск в отдельном окне без консоли

Просто дважды кликните по любому из этих файлов.

### Вариант 2: Через командную строку

```bash
cd путь\к\mod_load_order_manager
python main.py
```

Или:
```bash
python3 main.py
py main.py
```

### Вариант 3: Прямой запуск

Двойной клик по файлу `main.py` (если Python ассоциирован с `.py` файлами).

## Решение проблем

### Ошибка "command not recognized"

Если при запуске появляется ошибка о том, что команда не распознана:

1. Убедитесь, что Python установлен:
   - Откройте командную строку
   - Введите `python --version` или `py --version`
   - Если команда не работает, установите Python с [python.org](https://www.python.org/downloads/)

2. Добавьте Python в PATH:
   - При установке Python выберите опцию "Add Python to PATH"
   - Или добавьте путь к Python вручную в системные переменные

3. Используйте полный путь к Python:
   - Найдите где установлен Python (обычно `C:\Python3x\` или `C:\Users\ВашеИмя\AppData\Local\Programs\Python\`)
   - Отредактируйте bat-файл, заменив `python` на полный путь, например: `"C:\Python39\python.exe" main.py`

### Ошибка "No module named 'tkinter'"

Если Python установлен, но Tkinter отсутствует:
- **Windows**: Переустановите Python с официального сайта, выбрав опцию "tcl/tk and IDLE"
- **Linux**: `sudo apt-get install python3-tk`
- **macOS**: Tkinter обычно уже установлен
