# Third Person Light

Мод добавляет свет к камере третьего лица, аналогичный свету первого лица.

## Установка

Поместите папку `ThirdPersonLight` в директорию `darktide-mods`.

## Настройки

- **Включено** - Включить/выключить мод
- **Интенсивность света** - Интенсивность света (0.1 - 20.0)
- **Радиус света** - Радиус действия света (1.0 - 50.0)
- **Цвет света** - RGB компоненты цвета света (0-255)

## Использование

Мод автоматически активируется при переключении в третий режим камеры и создает свет на позиции камеры.
