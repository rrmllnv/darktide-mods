# EffectAggregate

Мод для агрегации и отображения информации об эффектах.

## Описание

EffectAggregate - мод для Warhammer 40,000: Darktide, который агрегирует и отображает информацию об эффектах.

## Установка

Скопируйте папку `EffectAggregate` в директорию модов Darktide Mod Framework.

## Настройки

Настройки мода доступны в меню настроек игры.
