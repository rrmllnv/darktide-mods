# Пути к иконкам в Darktide

## Что такое путь `content/ui/materials/hud/interactions/icons/default`?

Это **виртуальный путь** внутри игрового движка Darktide. Это не реальный путь к файлу на диске, а логический идентификатор ресурса в игровой системе.

## Где физически находятся файлы?

### 1. **Bundle файлы (запакованные архивы)**

Иконки и другие UI материалы хранятся в **bundle файлах** (`.bundle` или `.pak` файлы) в папке игры:

```
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\bundle\
```

или

```
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\data\
```

### 2. **Структура внутри bundle**

Внутри bundle файлов ресурсы организованы по виртуальным путям:
- `content/ui/materials/hud/interactions/icons/default` → находится в bundle файле
- Файл может иметь расширение `.dds`, `.png`, `.tga` или быть в формате материала движка

## Это не зашифровано, а запаковано!

- ❌ **Не зашифровано** - данные не защищены шифрованием
- ✅ **Запаковано/сжато** - файлы упакованы в архив для оптимизации
- ✅ **Можно распаковать** - нужны специальные инструменты

## Как найти/извлечь иконки?

### Вариант 1: Поиск в папке игры

Проверьте папки:
```
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\bundle\
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\data\
C:\Program Files (x86)\Steam\steamapps\common\Warhammer 40,000 DARKTIDE\assets\
```

### Вариант 2: Инструменты для распаковки

Для извлечения ресурсов из bundle файлов нужны специальные инструменты:
- **FModel** - универсальный инструмент для Unreal Engine игр
- **UModel** - для просмотра и извлечения ресурсов
- **Darktide-specific tools** - инструменты сообщества моддеров

### Вариант 3: Использование в коде мода

В коде мода вы просто используете виртуальный путь:

```lua
interaction_icon = "content/ui/materials/hud/interactions/icons/default"
```

Игровой движок сам найдет нужный ресурс в bundle файлах.

## Формат файлов

Иконки могут быть:
- **Материалы движка** (`.material` или встроенные в bundle)
- **Текстуры** (`.dds`, `.png`, `.tga`)
- **Векторные иконки** (`.svg` или векторные материалы)

## Важно

⚠️ **Не изменяйте bundle файлы напрямую!** Это может сломать игру.

✅ **Используйте моды** - моды могут переопределять пути к ресурсам или добавлять свои.
